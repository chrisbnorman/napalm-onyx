# napalm-onyx
Napalm driver targeting Mellanox devices.

Supported devices
This driver initially targets the Mellanox device with Onyx OS though other, similar, devices may be supported.

### Implemented APIs

* get_arp_table
* get_facts
* get_interfaces
* get_mac_address_table
* get_lldp_neighbors
* is_alive
* cli
